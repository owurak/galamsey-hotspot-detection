# Galamsey Hotspot Detection Dashboard

## 📊 Project Overview

This project analyzes illegal small-scale mining (galamsey) activities across Ghana using Power BI. The goal is to transform raw CSV data into actionable insights that help identify high-risk regions and support environmental monitoring efforts.

## 🚨 Problem Statement

Open Forest Watch Africa (OFWA) faces challenges in:

* Identifying galamsey hotspots quickly
* Extracting insights from raw data
* Sharing findings efficiently with stakeholders

## 🎯 Objectives

* Identify regions and cities with the highest galamsey activity
* Calculate total and average mining sites
* Detect high-risk cities based on thresholds
* Provide insights for better decision-making

## 📈 Key Insights

* Highlighted regions with the highest concentration of illegal mining sites
* Identified cities exceeding risk thresholds (e.g., >10 sites)
* Revealed patterns indicating emerging hotspots
* Enabled data-driven prioritization of intervention areas

## 🛠 Tools Used

* Power BI (Data visualization & dashboard creation)
* CSV dataset (Galamsey activity data)

## 💡 Business Impact

This dashboard helps:

* Authorities prioritize high-risk regions
* Improve resource allocation
* Monitor trends in illegal mining activities
* Support environmental protection initiatives
